# 2526_Maker_Flashlight
Objectif : Concevoir une lampe d'éclairage sur un vélo afin d'augmenter la sécutité en étant plus visible. 

### Cahier des charges : 
- 3 modes d’éclairages
- Étanche
- Bonus pour le rendre avancée : énergie solaire, allumage automatique (capteur luminosité)

### Planning prévisionnel : 
1. Etablir une analyse fonctionnelle et la BOM
2. Schématique et routage conception électronique
3. CAO pour le boitier
4. Assemblage et test
